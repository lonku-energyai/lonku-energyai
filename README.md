#### [Production Decline Analysis](https://github.com/lonku-energyai/production-decline-analysis) &nbsp;`🚧`
Decline-curve analysis on real field production data — fit Arps equations to daily oil/gas/water rates, estimate **EUR**, forecast remaining production, and flag wells declining faster than expected. End-to-end on messy, real-world data.
<br>**Tech:** `Python` · `pandas` · `SciPy` · `Plotly`

#### [Predictive Maintenance — Remaining Useful Life](https://github.com/lonku-energyai/rul-predictive-maintenance) &nbsp;`🚧`
Time-series machine learning to predict **Remaining Useful Life** from sensor streams — the same approach that flags impending failure on **pumps, compressors, and ESPs** in field operations, framed around downtime cost.
<br>**Tech:** `Python` · `scikit-learn` · `pandas`

#### [CFD Surrogate Model](https://github.com/lonku-energyai/cfd-surrogate-model) &nbsp;`🚧` &nbsp;⭐
A machine-learning surrogate trained on CFD results that predicts outputs — pressure drop, drag, heat transfer — **instantly, without running the full simulation**. Fuses CFD and ML for fast design exploration.
<br>**Tech:** `ANSYS Fluent` · `Python` · `scikit-learn`
